# Landing Page - Veterinária Dra. Karla

Landing page profissional desenvolvida para clínica veterinária especializada em **atendimento domiciliar**, com foco em conversão e experiência do usuário.

## Sobre o Projeto

Este projeto consiste em uma landing page moderna e responsiva para uma veterinária que oferece atendimento domiciliar. O site foi desenvolvido com foco em:

- **Conversão**: Design estratégico para captar leads através do WhatsApp
- **UX/UI**: Interface limpa e profissional baseada em princípios de design médico
- **Performance**: Código otimizado e semântico
- **Acessibilidade**: HTML5 semântico e navegação intuitiva

## Funcionalidades

- Hero Section impactante com destaque para atendimento domiciliar
- Seção sobre a veterinária com apresentação profissional
- Serviços organizados em cards interativos
- CTA estratégico para agendamento de consultas
- Credenciais e formação profissional
- Depoimentos de clientes satisfeitos
- Formulário de contato com integração direta ao WhatsApp
- Design responsivo para todos os dispositivos

## Tecnologias Utilizadas

- **React** - Biblioteca JavaScript para construção de interfaces
- **CSS3** - Estilização com CSS Modules e variáveis CSS
- **HTML5 Semântico** - Estrutura acessível e otimizada para SEO
- **WhatsApp API** - Integração para envio direto de mensagens

### Tipografia

- **Fonte Principal**: Inter, system fonts
- **Hierarquia**: Headers de 2rem a 3.75rem, body text 1rem a 1.125rem

## Responsividade

O projeto utiliza CSS Grid e Media Queries para garantir uma experiência otimizada em:

- **Mobile**: < 640px
- **Tablet**: 641px - 1024px
- **Desktop**: > 1024px

## SEO e Otimização para Buscas Locais

### Meta Tags Implementadas

O projeto já conta com estrutura semântica, mas para otimizar para buscas locais (Google):

1. **Google My Business** - Essencial para SEO local:
   - Criar perfil completo
   - Adicionar fotos profissionais
   - Coletar avaliações de clientes
   - Postar atualizações regulares

2. **Otimizações On-Page**:
   - URLs amigáveis
   - Alt text descritivo em todas as imagens
   - Heading hierarchy correta (H1, H2, H3)
   - Conteúdo com palavras-chave locais

3. **Performance**:
   - Imagens otimizadas (WebP)
   - Lazy loading
   - Minificação de CSS/JS
